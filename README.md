steps
===

1. installation
    + sudo update
    + sudo apt install python3-pip
    - sudo apt install python3-virtualenv
2. initialization
    - >virtualenv <virtual_env_name> 
    - source <virtual_env_name> /bin/activate
    - pip3 install -r requirement.txt
3. run app.py
